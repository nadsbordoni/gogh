# Projeto Gogh

## Descrição
Gogh é uma plataforma digital inovadora criada para conectar artistas locais a um público mais amplo. 
Seu objetivo é permitir que artistas independentes mostrem suas obras de arte de forma acessível, fortalecendo a economia criativa local. 
Além disso, o Gogh busca democratizar o acesso à arte, proporcionando um ambiente virtual onde artistas e colecionadores podem interagir diretamente.

## Vantagens do Gogh
- Valorização de Artistas Locais: O Gogh oferece um espaço dedicado para que artistas independentes possam divulgar e monetizar suas obras.
- Conexão Direta com o Público: Facilita a interação entre artistas e consumidores, permitindo vendas diretas e solicitações de encomendas personalizadas.
- Plataforma Acessível e Intuitiva: Com filtros de busca por tipo de arte, preço e localização, facilita a navegação para o público e aumenta a visibilidade dos artistas.
- Integração com Meios de Pagamento: A plataforma oferece formas seguras de pagamento, como PIX, cartão de crédito/débito, e boleto bancário.

## Linguagens e Estruturas Utilizadas
- **Linguagem**: Python

- **Estruturas**:

    - Arquivos .py contendo a lógica principal para a gestão de usuários, artistas, obras e pagamentos.
    - Utilização de funções para manipulação de dados, como CRUD (Create, Read, Update, Delete).
    - Estruturas de Dados: Listas, dicionários e tuplas para armazenar e organizar informações de artistas e suas obras.
